MUX Mail to BrandMail

From Gwen Morse's Wiki
Jump to: navigation, search 
@@ This allows those people used to BrandyMail commands to use them to write messages while on a
@@ MUX server that uses @mail. It's very handy to the dinos out there.

@@ I don't recall exactly where this came from, but, I *THINK* it was Pederson's Mux in a Minute.
@@ This would certainly be a reasonable guess, as Marvin Pederson wrote other mail functions to convert
@@ Mux mail to Brandy-mail commands.

@create Mux Mailer=10
&MAIL-SENDING Mux Mailer=$+mail *=*:@fo %#=@mail %0=%1
&MAIL-EDIT-TEXT Mux Mailer=$+edit text=%0/%1:@fo %#=@mail/edit %0=%1
&MAIL-EDIT-TO Mux Mailer=$+edit to=%0/%1:@fo %#=@mail/cc %0
&MAIL-EDIT-CC Mux Mailer=$+edit cc=%0/%1:@fo %#=@mail/cc %0
&MAIL-ABORT Mux Mailer=$+toss:@fo %#=@mail/abort
&MAIL-PROOF Mux Mailer=$+proof:@fo %#=@mail/proof
&MAIL-SEND Mux Mailer=$+send:@fo %#=@mail/send
&MAIL-SKIM Mux Mailer=$+skim:@fo %#=@mail
&MAIL-SKIM-RANGE Mux Mailer=$+skim *:@fo %#=@mail/list %0
&MAIL-READ-RANGE Mux Mailer=$+read *:@fo %#=@mail/read %0
&MAIL-HELP Mux Mailer=$+help mail:@fo %#=help @mail
&MAIL-HELP-TOPIC Mux Mailer=$+help mail *:@fo %#=help mail-%0
&MAIL-REVIEW Mux Mailer=$+review letters *=*:@fo %#=@mail/review %0=%1
&MAIL-RETRACT Mux Mailer=$+retract letter *=*:@fo %#=@mail/retract %0=%1
&MAIL-CLEAR Mux Mailer=$+clear *:@switch [get(%#/mail-option-delete)]=soft,{@fo %#=@mail/clear %0},hard,{@fo %#=@mail/clear %0;@fo %#=@mail/purge}
&MAIL-OPTIONS-DELETE Mux Mailer=$+set mail option delete=*:@switch %0=hard,{@fo %#=&mail-option-delete me=hard},soft,{&@fo %#=mail-option-delete me=soft}
&MAIL-DELETE Mux Mailer=$+delete *:@switch [get(%#/mail-option-delete)]=soft,{@fo %#=@mail/clear %)},hard,{@fo %#=@mail/clear %0;@fo %#=@mail/purge}
&MAIL-UNDELETE Mux Mailer=$+undelete *:@fo %#=@mail/unclear *
&MAIL-FLUSH Mux Mailer=$+flush:@fo %#=@mail/purge
&MAIL-MARK Mux Mailer=$+mark *:@fo %#=@mail/tag %0
&MAIL-UNMARK Mux Mailer=$+unmark *:@fo %#=@mail/untag %0
&MAIL-SAFE Mux Mailer=$+safe *:@fo %#=@mail/safe %0
&MAIL-ALIAS-PERSONAL Mux Mailer=$+add personal alias *=*:@fo %#=@malias *%0=%1
&MAIL-ALIAS-PERSONAL-LIST Mux Mailer=$+list personal aliases:@fo %#=@malias
&MAIL-ALIAS-PERSONAL-LIST-EXP Mux Mailer=$+expand personal alias *:@fo %#=@malias *%0
&MAIL-ALIAS-PERSONAL-REMOVE Mux Mailer=$+remove personal alias *:@fo %#=@malias/delete %0
&MAIL-ALIAS-GLOBAL-LIST Mux Mailer=$+list global aliases:@fo %#=@malias
&MAIL-ALIAS-GLOBAL-LIST-EXP Mux Mailer=$+expand global alias *:@fo %#=@malias *%0
&MAIL-ALIAS-GLOBAL-REMOVE Mux Mailer=$+remove global alias *:@fo %#=@malias/delete %0
&MAIL-FORWARD Mux Mailer=$+forward *=*:@fo %#=@mail/fwd %1=%0
&MAIL-NOTES Mux Mailer=$+note *=*:@fo %#=@mail/quick me/%0=%1
@set Mux Mailer=Inherit
@set Mux Mailer=!no_command
@set Mux Mailer=safe
