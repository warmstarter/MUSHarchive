@@ This stripper code was given to me by Cole, the original coder.
@@ It was meant to be shared at any games I wanted to give it to, I offered it to Metaire By Night, a game that never officially opened.
@@ Now I have it archived here.

@create Parent: Stripper=10
@Desc Parent: Stripper=%R%R%T[u(u(sex)-desc)]%R%R[center(Type 'Stripper-help' for help,79)]%R[switch(owner(me),%#,[center(Type 'Owner-help' for configuration help,79)])]
&MALE-DESC Parent: Stripper=This is the male description for the stripper, type &MALE-DESC [num(me)]=<description> to set my description.
&FEMALE-DESC Parent: Stripper=This is the female description for the stripper, type &FEMALE-DESC [num(me)]=<description> to set my description.
@Sex Parent: Stripper=Male
&CMD_STRIPPER_HELP Parent: Stripper=$STRIPPER-HELP:@pemit %#=%ch%crStripper Code %cw[u(Version)] %crby [u(Credits)]%cn%R%R[ljust(%chStripper-help%cn,20)]- This help menu.%R[ljust(%chtip <pose>%cn,20)]- Tip the stripper, with <pose>.%R[ljust(%chflirt <pose>%ch,20)]- Flirt with stripper, by using <pose>.%R%RExample of useage:%R%chtip%cn tucks a few bills under [name(me)]'s g-string and pats [switch(u(sex),MALE,his,FEMALE,her)] behind.%r%RThe stripper will pose.%R[name(%#)] tucks a few bills under [name(me)]'s g-string and pats [switch(u(sex),MALE,his,FEMALE,her)] behind.%R%RA few seconds after the initial command usage, the stripper will pose a reaction.%R
&VERSION Parent: Stripper=V2.0
&CREDITS Parent: Stripper=Cole@MedNights
&CMD_FLIRT Parent: Stripper=$flirt *:@emit [name(%#)] %0;@wait 5=@emit [u(FLIRT_[u(SEX)]_[u(%#/SEX)]_[u(Orientation)]_[inc(rand(words(lattr(me/FLIRT_[u(sex)]*))))])]
&CMD_TIP Parent: Stripper=$tip *:@emit [name(%#)] %0;@wait 5=@emit [u(TIP_[u(SEX)]_[u(%#/SEX)]_[u(Orientation)]_[inc(rand(words(lattr(me/TIP_[u(sex)]*))))])]
@Listen Parent: Stripper=*
@Ahear Parent: Stripper=@switch [match(%#,#6)][match([last(%0,%b)],arrived.)]=01,{@emit [u(POSE_[u(SEX)]_[inc(rand(words(lattr(me/POSE_[u(sex)]*))))])]},00,{@switch [gte([match(%0,*[name(me)]*)],1)]=1,{@emit [u(HEARD_NAME_[u(SEX)]_[u(%#/SEX)]_[inc(rand(words(lattr(me/HEARD_NAME_[u(sex)]_[u(%#/SEX)]*))))])]}}
&HEARD_NAME_MALE_MALE_1 Parent: Stripper=Random Response Male to Male Heard Name Used 1
&HEARD_NAME_FEMALE_MALE_1 Parent: Stripper=Random Response Female to Male Heard Name Used 1
&HEARD_NAME_MALE_FEMALE_1 Parent: Stripper=Random Response Male to Female Heard Name Used 1
&HEARD_NAME_FEMALE_FEMALE_1 Parent: Stripper=Random Response Female to Female Heard Name Used 1
&POSE_MALE_1 Parent: Stripper=Random Arrival Pose for Male Stripper 1
&POSE_FEMALE_1 Parent: Stripper=Random Arrival Pose for Female Stripper 1
&CMD_OWNER_HELP Parent: Stripper=$OWNER-HELP:@switch [owner(me)]=%#,{@pemit %#=%chOwner Help for Stripper %cn[u(version)]%ch by [u(Credits)]%cn%R%RThe following help topics are available for helping you create a customized stripper.%R%R%ch1%cn - Flirting Reactions.%R%ch2%cn - Tip reactions.%R%ch3%cn - Responses to hearing the strippers name.%R%ch4%cn - Arrival Poses.%R%ch5%cn - Gender Determinations.%R%ch6%cn - Descriptions.%R%ch7%cn - Setting attributes.%R%ch8%cn - Substitutions.%R%ch9%cn - Orientation options.%R%RType %chOwner-Help #%cn for details.}
&CMD_OWNER_HELP_OPTION Parent: Stripper=$OWNER-HELP *:@switch [owner(me)]=%#,{@pemit %#=%chOwner Help for Stripper%cn [u(Version)] %chby [u(Credits)]%cn%R%R[u(HELP_%0)]%R}
&HELP_1 Parent: Stripper=Configuring Flirting:%R%RTo configure the way the stripper responds to flirting, you may set the following attributes.%R%R[ljust(FLIRT_MALE_MALE_STRAIGHT_#,31)]- Straight Male Stripper to Male Player pose #%R[ljust(FLIRT_MALE_MALE_BI_#,31)]- Bi Male Stripper to Male Player pose #%R[ljust(FLIRT_MALE_MALE_GAY_#,31)]- Gay Male Stripper to Male Player pose #%R[ljust(FLIRT_MALE_FEMALE_STRAIGHT_#,31)]- Straight Male Stripper to Female Player pose #%R[ljust(FLIRT_MALE_FEMALE_BI_#,31)]- Bi Male Stripper to Female Player pose #%R[ljust(FLIRT_MALE_FEMALE_GAY_#,31)]- Gay Male Stripper to Female Player pose #%R[ljust(FLIRT_FEMALE_MALE_STRAIGHT_#,31)]- Straight Female Stripper to Male player pose #%R[ljust(FLIRT_FEMALE_MALE_BI_#,31)]- Bi Female Stripper to Male player pose #%R[ljust(FLIRT_FEMALE_MALE_GAY_#,31)]- Lesbian Female Stripper to Male player pose #%R[ljust(FLIRT_FEMALE_FEMALE_STRAIGHT_#,31)]- Straight Female Stripper to Female Player pose #%R[ljust(FLIRT_FEMALE_FEMALE_BI_#,31)]- Bi Female Stripper to Female Player pose #%R[ljust(FLIRT_FEMALE_FEMALE_GAY_#,31)]- Lesbian Female Stripper to Female Player pose #%R%RPlease note # can be anything from 1 up to the maximum number of attributes the object can hold. However, do not skip numbers. Be sure to enter them sequentially from 1 upward.
&HELP_# Parent: Stripper=Please use a number, 1 through 9 when using Owner-Help.
&HELP_2 Parent: Stripper=Configuring Tip Reactions:%R%RThe Tip reactions are configured the same way as the Flirt reactions, The following attributes are used.%R%R[ljust(TIP_MALE_MALE_STRAIGHT_#,29)]- Straight Male Stripper to Male Player pose #%R[ljust(TIP_MALE_MALE_BI_#,29)]- Bi Male Stripper to Male Player pose #%R[ljust(TIP_MALE_MALE_GAY_#,29)]- Gay Male Stripper to Male Player pose #%R[ljust(TIP_MALE_FEMALE_STRAIGHT_#,29)]- Straight Male Stripper to Female Player pose #%R[ljust(TIP_MALE_FEMALE_BI_#,29)]- Bi Male Stripper to Female Player pose #%R[ljust(TIP_MALE_FEMALE_GAY_#,29)]- Gay Male Stripper to Female Player pose #%R[ljust(TIP_FEMALE_MALE_STRAIGHT_#,29)]- Straight Female Stripper to Male Player Pose #%R[ljust(TIP_FEMALE_MALE_BI_#,29)]- Bi Female Stripper to Male Player Pose #%R[ljust(TIP_FEMALE_MALE_GAY_#,29)]- Lesbian Female Stripper to Male Player Pose #%R[ljust(TIP_FEMALE_FEMALE_STRAIGHT_#,29)]- Straight Female Stripper to Female Player pose #%R[ljust(TIP_FEMALE_FEMALE_BI_#,29)]- Bi Female Stripper to Female Player pose #%R[ljust(TIP_FEMALE_FEMALE_GAY_#,29)]- Lesbian Female Stripper to Female Player pose #%R%RPlease note # can be anything from 1 up to the maximum number of attributes the object can hold. However, do not skip numbers. Be sure to enter them sequentially from 1 upward.
&HELP_3 Parent: Stripper=Configuring Heard Name Responses:%R%RThe stripper responds to events where a player either poses about the stripper, or speaks the strippers name. To set possible poses, use the following attributes.%R%RHEARD_NAME_MALE_MALE_# - Male Stripper to Male player pose #%RHEARD_NAME_MALE_FEMALE_# - Male stripper to Female player pose #%RHEARD_NAME_FEMALE_MALE_# - Female Stripper to Male player pose #%RHEARD_NAME_FEMALE_FEMALE_# - Female Stripper to Female player pose #%R%RPlease note # can be anything from 1 up to the maximum number of attributes the object can hold. However, do not skip numbers. Be sure to enter them sequentially from 1 upward.
&HELP_4 Parent: Stripper=Arrival Pose Configuration:%R%RThe stripper responds to the 'Player has arrived.' announcement the mu* makes when a player enters the room. To configure the poses which the stripper makes, use the following attributes.%R%RPOSE_MALE_# - Male Stripper pose #%RPOSE_FEMALE_# - Female Stripper pose #%R%RPlease note, # can be anything from 1 up to the maximum number of attributes the object can hold. However, do not skip numbers. Be sure to enter them sequentially from 1 upward.
&HELP_5 Parent: Stripper=Gender Determinations:%R%RThe stripper code is designed to function both as male and female, as a result, it adapts it's self to it's current @sex setting.%R%RTo be sure your stripper functions properly, you must either set the stripper to be male or female with the following command.%R%R@sex <Striper name>=Male%Ror%R@sex <Stripper name>=Female%R%RIf you do not do this, the stripper will default to it's parents sex.%R
&HELP_6 Parent: Stripper=Descriptions:%R%RDO NOT @desc your stripper! Doing so will disable a portion of the gender determination system, and will also remove the automatic help display.%R%RTo set your strippers description use one of the following attributes, or both, if you plan to use the stripper to its full capabilities.%R%RMALE-DESC - The male stripper description.%RFEMALE-DESC - The female stripper description.%R%RNote you do not need to insert the first %%R and %%T set for the description as it will automatically be inserted, and a trailing %%R will be appended before the help information.
&HELP_7 Parent: Stripper=Setting attributes:%R%RAttributes are a way of storing information on an object, player, or room. System attributes are set with an @ prefix, where as custom attributes are set with an & prefix. You will be using both when customizing your stripper.%R%RUnless otherwise stated in these help files, use the & prefex when setting attributes. The format is as follows%R%R%ch%cg&%cb<ATTRIBUTE> %cr<object>%cy=%cc<Information>%cn%R%RFor expample.%R%R%ch%cg&%cbMY_SILLY_ATTRIBUTE%cr MyStripper%cy=%ccJust a bit of text.%cn%R%RThe above sets the attribute %ch%cbMY_SILLY_ATTRIBUTE%cn on %ch%crMyStripper%cn with the text of %ch%ccJust a bit of text.%cn
&HELP_8 Parent: Stripper=Substitutions:%R%RSubsitutions used for name and gender will not work in this system, however formatting substitutions will.%R%RTo insert the players name it a pose use \[name(%%#)\]%R%RFor more help with substitutions, see 'help function list' and 'help substitutions'
&HELP_9 Parent: Stripper=Orientation:%R%RBy request, the strippers now support a sexual orientation, this orientation is set with the &ORIENTATION attribute and can be one of the following.%R%R&ORIENTATION <name>=Straight%R&ORIENTATION <name>=Bi%R&ORIENTATION <name>=Gay%R%RThis setting determines how the stripper will react to players of their own gender, and players of opposite gender. It also offers a bit of flexibility by adding one more option in the poses.
&ORIENTATION Parent: Stripper=Gay
@set Parent: Stripper=VISUAL
@set Parent: Stripper=PARENT_OK
@set Parent: Stripper=HALTED
@set Parent: Stripper=SAFE
