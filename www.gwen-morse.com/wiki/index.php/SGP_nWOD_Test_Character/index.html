@@
@@  Genre-specific functions, like things that generate really special
@@  -ized display output.  Look at the other installers for clues.
@@

@@  This is a sample of an /quotable file for statting characters using SGP Cgen.  
@@ The main purpose of the file is to test your installation of a nWoD chargen.

@@ Change "Monkey" to the player name you'd like to stat up.
@@ Monkey is my Code Monkey, a test player I often create for code debugging.

+setstat Monkey/intelligence=2
+setstat Monkey/strength=2
+setstat Monkey/presence=2
+setstat Monkey/wits=2
+setstat Monkey/dexterity=2
+setstat Monkey/manipulation=2
+setstat Monkey/resolve=2
+setstat Monkey/perception=2
+setstat Monkey/composure=4

@@ Skills @@
@@ Mental, Physical, and Social @@
@@ &list-mental SGP Cgen: nWoD List Object=Academics|Computer|Crafts|Investigation|Medicine|Occult|Politics|Science
@@ &list-physical SGP Cgen: nWoD List Object=Athletics|Brawl|Drive|Firearms|Larceny|Stealth|Survival|Weaponry
@@ &list-social SGP Cgen: nWoD List Object=Animal Ken|Empathy|Expression|Intimidation|Persuasion|Socialize|Streetwise|Subterfuge

+setstat Monkey/academics=2
+setstat Monkey/medicine=2
+setstat Monkey/occult=2
+setstat Monkey/athletics=2
+setstat Monkey/firearms=2
+setstat Monkey/survival=2
+setstat Monkey/animal ken=2
+setstat Monkey/empathy=2
+setstat Monkey/socialize=2
+setstat Monkey/streetwise=1
+setstat Monkey/subterfuge=4
+setstat Monkey/expression=2
+setstat Monkey/intimidation=3

@@  Permanent Blood Pool Value

+setstat Monkey/blood=10

@@ Pools

+setstat Monkey/mana=7
+setstat Monkey/willpower=7
+setstat Monkey/gnosis=5
+setstat Monkey/willpower=7

@@ Death|Fate|Forces|Life|Matter|Mind|Prime|Space|Spirit|Time

+setstat Monkey/life=2
+setstat Monkey/prime=3
+setstat Monkey/death=1

@@ Check for exact format on race-names
@@ &SHEET-MAGE SGP Cgen: nWOD Support Code=@pemit %1=[center(<%b%b[name(%0)]%b%b>,78,=)]%R[u(#140/fn_topname,*%0)] DOB: [get(*%0/DOB)][u(fn-display-strings,%0,Shadowname:STRING-SHADOWNAME|Order:STRING-ORDER|Concept:STRING-CONCEPT|Path:STRING-PATH|Cabal:STRING-CABAL|Legacy:STRING-LEGACY)]%r[u(#140/fn_attributes,*%0)]%r[u(#140/fn_abilities,*%0)]%R[center(%bArcana%b,78,~)]%R[u(#140/fn_arcana,*%0)]%r[center(%bMiscellaneous%b,78,~)]%R[center(%bPools%b,78,~)]%R[u(#140/fn_pools,*%0)];@pemit %1=[u(#140/fn_common-lists,*%0)];@pemit %1=%r[u(#140/FN_HEALTH,%0)]%R[repeat(=,78)]

@@ All LIST-STRINGS entries which do not have fixed options: Cabal|Concept|Legend|Motley|Name|Nationality|Shadowname

&LIST-STRINGS SGP Cgen: nWOD List Object=Faction|Kith|Legacy|Order|Path|Race|Season|Seeming|Vice|Virtue|Cabal|Concept|Legend|Motley|Fullname|Nationality|Shadowname

@@ Faction @@
&LIST-FACTION SGP Cgen: nWOD List Object=Academic|Artist|Business|Crime|Faith|Government|High Society|Law|Media|Medicine|Naturalist|Occult|Professional|Science|Trades|Unemployed

@@ Legacy @@
&LIST-LEGACY SGP Cgen: nWOD List Object=Eleventh Question|Cryptologos

@@ Kith @@
&LIST-KITH SGP Cgen: nWOD List Object=Broadback|Hunterheart|Runnerswift|Skitterskulk|Steepscrambler|Swimmerskin|Venombite|Windwing|Cleareyes|Coldscale|Roteater|Truefriend|Antiquarian|Gravewight|Leechfinger|Mirrorskin|Tunnelgrub|Lurkglider|Palewraith|Whisperwisp|Airtouched|Earthbones|Fireheart|Manikin|Snowskin|Waterborn|Woodblood|Blightbent|Levinquick|Sandharrowed|Bright One|Dancer|Draconic|Flowering|Muse|Telluric|Treasured|Cyclopean|Farwalker|Gargantuan|Gristlegrinder|Stonebones|Water-Dweller|Bloodbrute|Corpsegrinder|Render|Troll|Artist|Brewer|Chatelaine|Chirurgeon|Oracle|Smith|Soldier|Woodwalker|Author|Drudge|Gameplayer|Miner|Gremlin

@@ Order @@
&LIST-ORDER SGP Cgen: nWOD List Object=Invisible College|Guardians of the Veil|Silver Ladder|Steel Legion|Wise Society

@@ Path @@
&LIST-PATH SGP Cgen: nWOD List Object=Acanthus|Mastigos|Moros|Obrimos|Thyrsus

@@ Race @@
&LIST-RACE SGP Cgen: nWOD List Object=mortal|sighted|mage|changeling

@@ Season and Court @@
&LIST-SEASON SGP Cgen: nWOD List Object=Autumn Moon|Spring Sun|Summer Sun|Winter Moon

@@ Seeming @@
&LIST-SEEMINGS SGP Cgen: nWOD List Object=Beast|Darkling|Elemental|Fairest|Ogre|Wizened


+setstring Monkey/race=mage
+setstring Monkey/virtue=temperance
+setstring Monkey/vice=envy
+setstring Monkey/shadowname=Jamra bint Fazil Mubarak "Il'al Gina" Most Capable Singer
+setstring Monkey/concept=MyConcept
+setstring Monkey/path=acanthus
+setstring Monkey/cabal=Witches of Eastwick
+setstring Monkey/order=Wise Society
+setstring Monkey/legacy=cryptologos


@@ The setting of specialties is optional and are here
@@ for convenience in loading advanced characters.
@@ This is a FAKE OUT - +setspeciality doesn't exist anywhere in the SGP release.

+setspecialty Monkey/subterfuge=Reading Micro-Expressions

@@ list attributes are in the form of | separated lists.

&list-merits *Monkey=stuff|stuff2|stuff3|stuff4|stuff5|stuff6|stuff7|stuff8
