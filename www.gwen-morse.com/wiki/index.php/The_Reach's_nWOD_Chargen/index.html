This is part 1 of the Reach Chargen System. Part 2 would be The Reach's nWOD Chargen Module

As per Soma@WORA: The attached code should get you the chargen any type of nWoD creature you want to create. It has commands in there for every major sourcebook and every minor one except for Skinthieves. If you install it and it doesn't work, let me know. There may be some stray functions that are not placed into the actual objects.
