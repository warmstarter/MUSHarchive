+Help System

This system provides you with a global +help system. All you have to do is add text for the various help files, and you're off to the races.

Features

  • A complete family of +help commands, including the basic index (+help), the detailed help file view (+help ), and the search facility (+help/search).
  • A staff help family of commands (+shelp) for your wizard/staff/admin-only commands.
  • Help files can be stored so they aren't displayed in the basic index. That way files like +help +Finger2 don't clutter up your main page.

Platform

This code is designed for PennMUSH only. As long as PennMUSH stays backwards compatible, it will work on version 1.7.2 patchlevel 19 or higher. You're welcome to try it on older versions of Penn, or on TinyMUX, but I make no guarantees that it will work.

Code

Installer - Save this file and /quote it to your MUSH to install the +help system. Read the file for installation and setup notes.

Information about each object is contained in the object's description. Once the system is installed, look at the objects to see what you can configure.
