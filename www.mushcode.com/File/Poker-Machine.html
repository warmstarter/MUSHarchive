Poker Machine

A poker machine with ANSI graphics. (This poker machine uses a Role Playing MUSH money system, and the &VAR_MONEY has to be changed accordingly. With a little work it could be made to work using the built-in MUSH currency.)

Author: Demonic
Category: Games
Commands: @aenter, @aleave, @create, @describe, @enter, @failure, @idescribe, @leave, @oenter, @oleave, @oxenter, @oxleave, @parent, @set, @switch.
Functions: add(), after(), and(), ansi(), before(), center(), dec(), default(), edit(), eq(), extract(), gt(), gte(), hasflag(), hastype(), ifelse(), inc(), isdbref(), iter(), last(), ljust(), lnum(), loc(), lock(), lt(), lte(), match(), member(), mid(), mul(), name(), not(), num(), or(), pemit(), pmatch(), pos(), poss(), remit(), repeat(), replace(), rjust(), secure(), set(), setq(), setr(), setunion(), shuffle(), sort(), space(), sub(), switch(), trunc(), u(), v(), words(), xor().
Compatibility: CobraMUSH, PennMUSH, TinyBit, TinyMUX.
