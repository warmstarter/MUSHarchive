Natalia's Card Table

A multiplayer card table.

Author: Natalia@Full Circle MUX
Category: Games
Commands: @create, @describe, @dolist, @lock, @pemit, @switch.
Functions: add(), and(), capstr(), extract(), gte(), hasflag(), isnum(), loc(), lte(), match(), member(), name(), num(), remove(), repeat(), shuffle(), u().
Features: color substitutions.
Compatibility: TinyMUX.

Instructions

Copy and paste the below code into a compatible MUSH or MUX.
