Slot Machine

A slot machine with ANSI graphics. This slot machine uses a Role Playing MUSH money system, and the &VAR_MONEY has to be changed accordingly. With a little work it could be made to work using the built-in MUSH currency.

Author: Demonic
Category: Games
Commands: @aenter, @create, @describe, @enter, @failure, @idescribe, @leave, @oenter, @oleave, @oxenter, @oxleave, @parent, @set, @switch.
Functions: add(), after(), and(), ansi(), center(), dec(), edit(), eq(), extract(), first(), gt(), gte(), hastype(), if(), ifelse(), inc(), isdbref(), iter(), last(), lattr(), lcstr(), left(), ljust(), lnum(), loc(), lock(), lte(), match(), matchall(), member(), mid(), mul(), munge(), name(), not(), num(), oemit(), or(), orflags(), parent(), pemit(), rand(), remit(), remove(), repeat(), revwords(), rjust(), secure(), set(), setq(), setr(), setunion(), sort(), space(), strlen(), sub(), switch(), trunc(), u(), v(), words(), xget().
Compatibility: CobraMUSH, PennMUSH, TinyBit, TinyMUX.
