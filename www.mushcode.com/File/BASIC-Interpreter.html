BASIC Interpreter

A BASIC interpreter written in MUSHCode.

Author: Walker@M*U*S*H
Category: Other
Commands: @assert, @break, @create, @emit, @include, @link, @lock, @lset, @notify, @parent, @pemit, @retry, @set, @trigger, @wait.
