Big Banner

Just a toy to print out Banners.

Author: Satori@GlobalMUSH
Category: Fonts
Commands: @create, @describe, @lock.
Functions: add(), edit(), extract(), iter(), lnum(), mid(), r(), setq(), strlen(), u(), v().
Compatibility: CobraMUSH, PennMUSH, RhostMUSH, TinyBit, TinyMUSH, TinyMUX.

Instructions

Copy and paste the below code into a compatible MUSH or MUX.

MUSHCode for Big Banner

Select Code
@create BigBan
@lock BigBan==me
@Desc BigBan=Just a toy to print out Banners.
&LINE BigBan=[setq(0,iter(lnum(strlen(%0)),mid(%0,##,1)))][iter(lnum(7),%r[u(scan,##)])]
&SCAN BigBan=[iter(r(0),edit(extract(v(char_##),add(1,%0),1),-,%B))]
&BANNER BigBan=[iter(%0,u(line,##)%R)]
