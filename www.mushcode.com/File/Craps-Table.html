Craps Table

A craps table with ANSI graphics. (This craps table uses a Role Playing MUSH money system, and the &VAR_MONEY has to be changed accordingly. With a little work it could be made to work using the built-in MUSH currency.)

Author: Demonic
Category: Games
Commands: @aenter, @aleave, @create, @describe, @dolist, @enter, @failure, @idescribe, @leave, @oenter, @oleave, @oxenter, @oxleave, @parent, @pemit, @set, @switch, @trigger, @wait.
