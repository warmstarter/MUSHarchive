Trispis' Fireworks

An ASCII Art fireworks display for the Fourth of July.

Author: Trispis@M*U*S*H
Category: Other
Commands: @create, @dolist, @lock, @pemit, @set, @wait.
Functions: add(), ansi(), center(), inc(), iter(), lnum(), mul(), repeat(), space(), u().
Compatibility: CobraMUSH, PennMUSH, RhostMUSH, TinyBit, TinyMUSH, TinyMUX.

Instructions

Copy and paste the below code into a compatible MUSH or MUX.

MUSHCode for Trispis' Fireworks

Select Code
@create fireworks
@lock fireworks==me
@set fireworks=!NO_COMMAND
&2 fireworks=[space(2)]
&3 fireworks=[space(3)]
&4 fireworks=[space(4)]
&5 fireworks=[space(5)]
&6 fireworks=[space(6)]
&7 fireworks=[space(7)]
&8 fireworks=[space(8)]
