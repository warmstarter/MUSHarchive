Voting Booth

A global voting system. Options can be set on polls, such as A for yes, B for no, or A for yes, B for maybe, C for no, and so on.

Author: Demonic
Category: Globals
Commands: @aconnect, @create, @dolist, @force, @pemit, @set, @switch, @teleport, @wait.
Functions: add(), and(), ansi(), cat(), center(), convsecs(), edit(), eq(), extract(), first(), gt(), gte(), hasflag(), haspower(), hastype(), if(), ifelse(), inc(), isdbref(), iter(), last(), lattr(), lcstr(), ldelete(), left(), ljust(), lt(), lwho(), match(), member(), mid(), name(), not(), num(), or(), pemit(), repeat(), replace(), rest(), rjust(), secs(), secure(), set(), setdiff(), setr(), space(), squish(), strcat(), sub(), switch(), t(), trim(), trunc(), u(), ucstr(), v(), wipe(), words(), xget().
Compatibility: CobraMUSH, PennMUSH, RhostMUSH, TinyBit, TinyMUX.
