Dancing Flower in a Plastic Pot

This is a simple object that demonstrates how @listen, @ahear, @desc and @fail work.

Author: TIM's Mother@TinyTIM
Category: Other
Commands: @ahear, @create, @describe, @failure, @listen, @set, @teleport, @va, @xz.
Compatibility: CobraMUSH, PennMUSH, TinyBit, TinyTIM.

Instructions

Copy and paste the below code into a compatible MUSH or MUX.

MUSHCode for Dancing Flower in a Plastic Pot

Select Code
@create Dancing Flower in a Plastic Pot
@desc Dancing Flower in a Plastic Pot=It's a little plastic flower, wearing shades, in a little pot. It's standing by itself on the windowsill. If you YELL anything, it'll probably dance around.
End of decompile.
@set Dancing Flower in a Plastic Pot=X
@fail Dancing Flower in a Plastic Pot=You go to pick up the flower, but its such a stupid thing, you leave it alone, lest you be seen with it.
@va Dancing Flower in a Plastic Pot=EXPLANATION: This flower listens for anything with the word YELLS in it. Typing YELL FLOWER!!!! would make it run the AHEAR. Notice the use of the * to say "Any characters".
@xz Dancing Flower in a Plastic Pot=FOR MORE INFORMATION: Check these Help Files: HELP LISTEN, HELP AHEAR, HELP @DESC, HELP FAIL
@listen Dancing Flower in a Plastic Pot=*yells*
@ahear Dancing Flower in a Plastic Pot=:dances around a little. You hear a little whirring. Then it stops.
