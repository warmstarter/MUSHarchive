Dyason's Social Commands

A collection of simple social commands.

Author: Dyason@GlobalMUSH
Category: Other
Commands: @create, @remit, @set.
Functions: ansi(), name().
Compatibility: CobraMUSH, PennMUSH, TinyBit, TinyMUX.

Instructions

Copy and paste the below code into a compatible MUSH or MUX.

MUSHCode for Dyason's Social Commands

Select Code
@create Commands
&BOUNCE0 Commands=$bounce *:@remit %L=[ansi(yh,%N)] [ansi(ch,bounces %0)]
