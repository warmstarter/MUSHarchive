Ian's Text

An original ascii art font, with upper and lower case letters, numbers and symbols.

Author: Ian@BrazilMUX
Category: Fonts
Commands: @create, @lock, @pemit, @set.
Functions: extract(), foreach(), pos(), setq(), setr(), v().
Compatibility: CobraMUSH, PennMUSH, RhostMUSH, TinyBit, TinyMUSH, TinyMUX.

Instructions

Copy and paste the below code into a compatible MUSH or MUX.

@create Text
@lock Text==me
@set Text=!NO_COMMAND
@lock/user:Basic Text==#0
