PennMUSH Function Library 1.0

The Function Library is a group of objects with a bunch of (Mostly and hopefully) useful ufuns and other softcode.

Author: Raevnos
Category: Functions
Commands: @create, @describe, @force, @pemit, @set, @teleport, @xa, @xb, @xc.
Functions: acos(), add(), alphamin(), andflags(), atan(), band(), bnand(), bor(), cand(), capstr(), ceil(), comp(), conn(), cos(), dec(), delete(), div(), edit(), elements(), eq(), extract(), fdiv(), filter(), first(), fold(), foreach(), grab(), graball(), gt(), gte(), hasflag(), haspower(), hastype(), if(), inc(), insert(), isint(), isnum(), isword(), last(), lcstr(), lit(), lmath(), ln(), lnum(), loc(), lt(), lte(), map(), match(), mean(), member(), mid(), min(), mod(), mul(), munge(), name(), not(), num(), obj(), orflags(), owner(), pi(), pos(), power(), regmatch(), repeat(), replace(), rest(), revwords(), right(), setinter(), setq(), setr(), shl(), shr(), shuffle(), sign(), sin(), sort(), space(), splice(), squish(), stripansi(), strlen(), strmatch(), sub(), switch(), t(), tagwrap(), trim(), type(), u(), ucstr(), ulocal(), v(), vadd(), vmag(), vmul(), wordpos(), words().
Compatibility: CobraMUSH, PennMUSH, TinyBit.
