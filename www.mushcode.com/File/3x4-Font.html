3x4 Font

A 3x4 font with letters and numbers, stored in attributes.

Author: RedWolf@CaveMUSH
Category: Fonts
Commands: @create, @describe, @lock.
Compatibility: PennMUSH, TinyBit, TinyMUSH, TinyMUX.

Instructions

Copy and paste the below code into a compatible MUSH or MUX.

MUSHCode for 3x4 Font

Select Code
@create 3x4 Font
@lock 3x4 Font=*RedWolf
&CREATIONDATE 3x4 Font=Tue Nov 9 17:24:43 2010 - By: Headset(#18) - Owned by: RedWolf(#2)
