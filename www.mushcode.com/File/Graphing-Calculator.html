Graphing Calculator

I know what you're thinking. A 'graphing calculator' on a text-based MUSH? This was designed to accept human style input (i.e: "graph x + sin(x) - 2x^2") and plot it on the graph. It can also be used to do simple and quick calculations. A 'math chip' is included that contains only the math calculation functions.

Author: Walker@M*U*S*H
Category: Other
Commands: @create, @lock, @parent, @pemit, @set.
Functions: abs(), acos(), add(), after(), align(), ansi(), asin(), atan(), atan2(), attrib_set(), before(), bound(), cand(), ceil(), chr(), cos(), dec(), die(), e(), edit(), elements(), extract(), fdiv(), first(), firstof(), floor(), fold(), gt(), hasattrp(), if(), isnum(), iter(), left(), lit(), ljust(), lmath(), ln(), lnum(), localize(), log(), lt(), map(), member(), mid(), mod(), mul(), name(), null(), pemit(), pi(), pos(), power(), rand(), regeditall(), regmatch(), repeat(), rest(), reswitch(), revwords(), round(), s(), setq(), setr(), sin(), space(), sqrt(), squish(), step(), strlen(), strmatch(), strreplace(), sub(), switch(), t(), tagwrap(), tan(), u(), v(), wordpos(), words().
Features: regexp commands.
Compatibility: PennMUSH.

Instructions

Copy and paste the below code into a compatible MUSH or MUX.
