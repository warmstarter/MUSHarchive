Coyote Clock

A clock with four different faces, and the ability to offset the time by any number of minutes (can be used to display time in a certain timezone).

Author: Coyote@TinyTIM
Category: Time
Commands: @create, @describe, @pemit, @set.
Functions: add(), convsecs(), edit(), extract(), fdiv(), fold(), gt(), gte(), lt(), lte(), mid(), mul(), replace(), round(), secs(), setq(), strlen(), sub(), switch(), u(), v().
Compatibility: CobraMUSH, PennMUSH, RhostMUSH, TinyBit, TinyMUSH, TinyMUX.

Instructions

Copy and paste the below code into a compatible MUSH or MUX.
